Qubit Module

class qubit.Qubit(alpha, beta)

Bases: object

A class to represent a quantum bit (Qubit) with two possible states: |0⟩ and |1⟩, described by complex amplitudes alpha and beta.

Attributes

alphafloat

The amplitude of the |0⟩ state, a value between 0 and 1.

betafloat

The amplitude of the |1⟩ state, a value between 0 and 1.

Methods

get_alpha():

Returns the amplitude of the |0⟩ state (alpha).

get_beta():

Returns the amplitude of the |1⟩ state (beta).

set_alpha(alpha):

Sets the amplitude of the |0⟩ state, ensuring it is between 0 and 1.

set_beta(beta):

Sets the amplitude of the |1⟩ state, ensuring it is between 0 and 1.
