Gate Class

class gate.Gate

Bases: object

Class to represent quantum gate. The user can choose a single qubit gate or a controlled qubit gate

Print the gate matrix with proper formatting, removing apostrophes. The matrix is displayed with 2 decimal precision, and values close to zero are rounded.
